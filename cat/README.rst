ft_cat
======

Concatenate files and print on the standard output.

``cat [OPTION]... [FILE]...``

DESC
----

With no File or when FILE is -, read standard input.


OPTIONS
^^^^^^^

- ``-e``
   equivalent to ``-vE``

- ``-v, --show-nonprinting``
   use ^ and M- notation, except for LFD and TAB.

- ``-E, --show-ends``
   display $ at end of each line.
